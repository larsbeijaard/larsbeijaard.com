Poller

A Discord bot that handles polls.

The Project

I'd always wanted to make a Discord bot, but didn't know what or how. Corona gave me the opportunity to think about it and let me do research on how to setup a Discord bot and what you all needed in order to make one. And as every new thing you learn, you create a small project, meaning that I will make another Discord bot, but more complex and extended.

What have I learnt?

I have learnt how to setup and how to make a Discord bot that can listen to commands and handle them. I used the C# language, because C# felt already comfortable for me. The things I needed to learn were, C# Discord API documentation which you can find here. And how to setup sockets using the Discord API

Languages & Information

  • Software: Visual Studio
